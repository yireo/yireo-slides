- Used by MFTF
    - Magento Functional Testing Framework

---
# Webdriver API
- Part of W3C specs
    - https://www.w3.org/TR/webdriver1/
- Different engines for different browsers
    - ChromeDriver
    - geckodriver
    - Edge webdriver
    - OperaChromiumDriver
- Different adapters for different languages
    - PHP, python, Java, C#, Ruby, JavaScript, ...
